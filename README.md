# PF2E Critical Forge: Arsenal

Additional card packs for **PF2E Critical Forge**.

## Current development pack

**Critical Forge Expanded** currently contains one deliberately simple test card:

- **Opening in the Guard / Lücke in der Deckung**
- Critical hit
- Slashing damage filter
- Applies **off-guard** to the target for 1 round

The pack is registered with `enabled: false`. It should therefore be visible as a protected extension pack in the Critical Forge Card Pack Editor without joining normal card selection until activated.

## Requirements

- Foundry Virtual Tabletop 14
- PF2e
- PF2E Critical Forge 0.8.0-rc.2 or newer
